MarkDown语法
MarkDown
基础语法
发布日期:   2019-07-08
文章字数:   1.6k
阅读时长:   6 分
阅读次数:  

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档。
Markdown 语言在 2004 由约翰·格鲁伯（英语：John Gruber）创建。
Markdown 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。
Markdown 编写的文档后缀为 .md, .markdown。

一.标题

用1-6个#代表html中<h1>-<h6>标题样式,该主题中对标题样式有修改

代码示例

    # 标题1                 
    ## 标题2               
    ### 标题3            
    #### 标题4              
    ##### 标题5
    ###### 标题6

二.段落

段落不需要特殊的标志，以两个空格加上回车表示换行

段落一
段落二（没加空格）
段落三（加空格）

代码示例

    段落一
    段落二（没加空格）  
    段落三（加空格）

三.字体

markdown中带有默认的字体样式，但改变颜色，字体，大小,位置等需要自己写html代码标签

斜体文本
粗体文本
粗体文本

粗斜体文本

代码示例

    _斜体文本_

    ***粗斜体文本***
    <font size=6 color=red face="楷体">___粗斜体文本___  </font>

四.分割线

可以用至少三个个星号，减号等建立一个分割线，个人感觉还是很人性化的

代码示例

    -----------
    **************

五.删除线，下划线

删除线在文字上添加中划线，也就是删除线在文字前后添加两个~~即可
下划线是通过html<u>标签来实现的

删除线
下划线

代码示例

    <u>下划线</u>

    [^3]: 脚注内容

九.代码

如果单个代码片段可以使用 ` 反引号把它包起来，代码区块使用4个空格，即一个制表符（Tab），或者一对 ``` 标签将代码包起来并可以指定语言

代码片段
System.out.print("hello world")

代码区块 

public static void main(String[] args){
    System.out.print("hello world");
}
 public static void main(String[] args){
        System.out.print("hello world");
    }

代码示例

    代码片段  
    `System.out.print("hello world")`  

    代码区块  

        public static void main(String[] args){
            System.out.print("hello world");
        }

    ```java
    public static void main(String[] args){
            System.out.print("hello world");
        }
    ```

十.链接

超链接使用[连接名称](连接地址)或者<连接地址>来实现html中<a></a>标签的使用
可以结合变量附带连接类似于脚注

这是一个连接我的博客

http://www.fbxblog.xyz

这个是变量1的连接1
这个是变量2的连接2

代码示例 

    这是一个连接[我的博客](http://www.fbxblog.xyz)

    <http://www.fbxblog.xyz>

    这个是变量1的连接[google]
    这个是变量2的连接[baidu]

  [google]: http://google.com/        "Google" 
  [baidu]: http://baidu.com/         "Baidu"

十一.图片

图片格式为![alt 属性文本](图片地址 "标题"),属性文本位置为图片加载失败时显示内容,”标题”为鼠标接触时的提示信息

也可用变量来表示地址值

如果想要指定宽高就使用<img>标签吧

代码示例

![](https://www.baidu.com/img/bd_logo1.png "百度搜索")

![][addr]
[addr]:https://www.baidu.com/img/bd_logo1.png "百度搜索"

<img src="https://www.baidu.com/img/bd_logo1.png" width="50%" height="50%" title="百度搜索">

表格的语法使用 | 来创建一个一个的单元格使用 - 来将表头展现出来,使用 : 符号来对该列的对齐方式, : 所在侧即为向哪侧对齐

左对齐 右对齐 居中
左对齐 右对齐 居中
左对齐 右对齐 居中
左对齐 右对齐 居中

代码示例

    表头1|表头2|表头3

    左对齐|右对齐|居中
